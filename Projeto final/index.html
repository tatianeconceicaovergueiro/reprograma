Equipe

Como diria Raul Seixas, sonho que se sonha só é só um sonho que se sonha só. Mas sonho que se sonha junto é realidade. O projeto surgiu quando quatro pessoas, que acreditam que a Constituição é a chave para a sociedade brasileira ser mais democrática, decidiram se juntar e sonhar juntas. Na foto, da esquerda para a direita: Gilberto Silveira; Tatiane Conceição Vergueiro; Aline Freitas e Tiago Firmino.
